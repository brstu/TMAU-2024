# Описание классов программы

## Класс `Model`

### Сводка
| Член                         | Описание                                                                 |
|------------------------------|--------------------------------------------------------------------------|
| `virtual double simulate(double prevOutput, double input, double prevInput) = 0` | Абстрактный метод для симуляции выхода модели.                          |
| `virtual ~Model() = default` | Деструктор для уничтожения объекта `Model`.                             |

### Описание
Класс `Model` является абстрактным классом, который задает интерфейс для всех моделей. Метод `simulate` должен быть реализован в производных классах.

---

## Класс `LinearModel`

### Сводка
| Член                         | Описание                                                                 |
|------------------------------|--------------------------------------------------------------------------|
| `public inline LinearModel(double a, double b)` | Конструктор для создания нового объекта `LinearModel`.                   |
| `public inline virtual double simulate(double prevOutput, double input, double prevInput)` | Переопределенный метод для расчета выходного значения.                   |
| `public virtual ~LinearModel() = default` | Деструктор для уничтожения объекта `LinearModel`.                       |
| `private double a`           | Параметр `a`.                                                           |
| `private double b`           | Параметр `b`.                                                           |

### Описание
Класс `LinearModel` представляет собой линейную модель управления, использующую линейную комбинацию предыдущего выхода и текущего входа для вычисления нового выхода.

---

## Класс `NonlinearModel`

### Сводка
| Член                         | Описание                                                                 |
|------------------------------|--------------------------------------------------------------------------|
| `public inline NonlinearModel(double a, double b, double c, double d)` | Конструктор для создания нового объекта `NonlinearModel`.               |
| `public inline virtual double simulate(double prevOutput, double input, double prevInput)` | Переопределенный метод для расчета выходного значения.                   |
| `public virtual ~NonlinearModel() = default` | Деструктор для уничтожения объекта `NonlinearModel`.                   |
| `private double a`           | Параметр `a`.                                                           |
| `private double b`           | Параметр `b`.                                                           |
| `private double c`           | Параметр `c`.                                                           |
| `private double d`           | Параметр `d`.                                                           |
| `private double prevOutput`   | Предыдущее значение выхода.                                             |
| `private double prevInput`    | Предыдущее значение входа.                                             |

### Описание
Класс `NonlinearModel` реализует нелинейную модель управления, учитывающую квадратичные и синусоидальные зависимости для вычисления выхода.

---

## Класс `PIDController`

### Сводка
| Член                         | Описание                                                                 |
|------------------------------|--------------------------------------------------------------------------|
| `public inline PIDController(double Kp, double Ki, double Kd)` | Конструктор для создания нового объекта `PIDController`.                |
| `public double compute(double setpoint, double measured)` | Метод для вычисления управляющего воздействия.                          |
| `public virtual ~PIDController() = default` | Деструктор для уничтожения объекта `PIDController`.                    |

### Описание
Класс `PIDController` реализует ПИД-регулятор для вычисления управляющего воздействия на основе заданной и измеренной величины.

---

## Класс `Simulation`

### Сводка
| Член                         | Описание                                                                 |
|------------------------------|--------------------------------------------------------------------------|
| `public inline Simulation(Model& model, PIDController& controller)` | Конструктор для создания нового объекта `Simulation`.                   |
| `public void run(int timeSteps, double setpoint)` | Метод для выполнения симуляции.                                         |

### Описание
Класс `Simulation` отвечает за выполнение симуляции, используя модель и контроллер для вычисления выходных значений и вывода результатов на экран.

---

## Заключение

Эти классы составляют основу программы, реализующей модели управления и симуляции. Они позволяют пользователю проводить анализ и тестирование различных подходов к управлению в системах.
